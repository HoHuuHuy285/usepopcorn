## Feature
- Tìm kiếm phim từ OMDb API ( [The Open Movie Database](https://www.omdbapi.com/) )
- Lưu lại những bộ phim mình đã coi ( LocalStorage )
- Đánh giá phim theo cảm nhận của bạn ( có sự so sánh với đánh giá của OMDb )
